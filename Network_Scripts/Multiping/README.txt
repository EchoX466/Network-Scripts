***** multiping.ps1 ***** 

=========================

This script performs an on-demand multi-host ping test, where the user supplies the target IPs at runtime instead of relying on a hardcoded list.

This script is similar to the heartbeat.ps1 script in that this accepts dynamic input from the user for a interactive runtime experience for momentary troubleshooting.
