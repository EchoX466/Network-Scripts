***** Heartbeat.ps1 *****

=========================

This script performs a one-time connectivity check (ping) against a list of computers/IPs and reports their reachability status, response time, and interpreted status code.


It’s essentially a lightweight network heartbeat monitor to verify whether systems are reachable at a given moment. 
I would expect this script to be scheduled and ran periodically to monitor equipments uptime and alert for outages. 

** This is still being worked on - A fair amount of improvements/changes can be made.
